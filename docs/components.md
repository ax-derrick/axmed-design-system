# Component Reference

Quick-reference index for all components in the Axmed Design System. Click any component name to view its full guide with props, examples, and patterns.

| Category | Component | Description |
|----------|-----------|-------------|
| **Foundations** | [AxText](components/ax-text.md) | Typography system with 12 variants, weights, colors, and ellipsis |
| | [AxBrand](components/ax-brand.md) | Axmed logo — icon or wordmark, light/dark themes |
| **Layout** | [AxCard](components/ax-card.md) | Content container with title, description, footer, and noPadding mode |
| | [AxDataCard](components/ax-card.md#axdatacard) | Compact label-value card with expand/collapse (subcomponent of AxCard) |
| | [AxSideNav](components/ax-side-nav.md) | Collapsible sidebar navigation with groups, sections, and mobile tab bar |
| | [AxHeader](components/ax-header.md) | App header with sidebar toggle, nav items, and branded variant |
| **Navigation** | [AxSteps](components/ax-steps.md) | Step indicator for multi-step flows and order tracking |
| **Actions** | [AxButton](components/ax-button.md) | Button with 6 variants — primary, secondary, ghost, danger, link, text |
| | [AxActionMenu](components/ax-action-menu.md) | Dropdown action menu for table rows and contextual actions |
| **Controls** | [AxInput](components/ax-input.md) | Text input with label, hint, error, and search subcomponent |
| | [AxInputOTP](components/ax-input-otp.md) | One-time password input with masking and length options |
| | [AxFilterBar](components/ax-filter-bar.md) | Search + filters + sort bar for tables and lists |
| | [AxUploader](components/ax-uploader.md) | Drag-and-drop file upload with size validation |
| | [AxFileItem](components/ax-file-item.md) | File row with status, progress, preview, and actions |
| **Data Display** | [AxTable](components/ax-table.md) | Data table with row states, mobile card view, skeleton, and cell components |
| | [AxTag](components/ax-tag.md) | Status label with tones, dots, pills, and country tag subcomponent |
| | [AxCompanyLogo](components/ax-company-logo.md) | Company avatar with initials fallback and deterministic colors |
| | [AxBadge](components/ax-badge.md) | Notification count badge with tone variants |
| | [AxStatCard](components/ax-stat-card.md) | KPI metric card with trend indicator and loading state |
| **Overlays** | [AxModal](components/ax-modal.md) | Dialog with size presets, danger mode, and description |
| | [AxDrawer](components/ax-drawer.md) | Slide-in panel with size presets and loading state |
| **Feedback** | [AxEmptyState](components/ax-empty-state.md) | Empty state placeholder with illustration and action |

## Quick Install

```bash
npm install axmed-design-system
```

```tsx
// app/layout.tsx
import "axmed-design-system/style.css";
```
